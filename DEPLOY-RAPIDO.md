# ⚡ Deploy Rápido - 5 Minutos

## 🎯 Opção Mais Rápida (Sem GitHub)

### 1️⃣ Criar Conta
- Acesse: https://dash.cloudflare.com/sign-up
- Crie conta gratuita
- Confirme email

### 2️⃣ Acessar Pages
- Login: https://dash.cloudflare.com/
- Menu lateral → **Workers & Pages**
- Botão **Create application**
- Escolha **Pages**
- Clique **Upload assets**

### 3️⃣ Fazer Upload
Arraste estes 5 arquivos:
```
✅ lp-hot-adult-v3.html
✅ document4960953111125427748_7ca098e6.mp4
✅ index.html
✅ _headers
✅ _redirects
```

### 4️⃣ Configurar
- **Project name**: `acesso-vip` (ou outro nome)
- Clique **Deploy site**
- Aguarde 30 segundos

### 5️⃣ Pronto! 🎉
Seu link será algo como:
```
https://acesso-vip.pages.dev
```
